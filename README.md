npm commands

In backend folder

Node js install packages
  > npm init

To create api packages
  > npm install express

Nodemon package
  > npm install nodemon

For env file
  > npm install dotenv

Encrytion usage
  > npm install bcryptjs

For json web token
  > npm i jsonwebtoken

In frontend - movies folder

  > npx create-react-app movies

Material UI
  > npm install @mui/material @emotion/react @emotionstyled
  
Material UI Icons
  > npm i @mui/icons-material

Make req to server - Axios
  > npm i axios

Router
  > npm i react-router-dom

Redux Tool Kit - Redux State Management
  > npm install @reduxjs/toolkit react-redux
